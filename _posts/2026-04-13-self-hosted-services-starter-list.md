If you are building a first homelab, start with the services that save time right away.

## Good first picks
- DNS filtering
- File sync
- Backup target
- Monitoring
- Media storage

## What to skip for now
Skip the complicated, noisy stuff until the machine is stable. The goal is a small server that feels useful, not a project that eats evenings.

## My rule
If a service does not make your life easier this week, it can wait.
